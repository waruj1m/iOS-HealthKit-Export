# iOS Health Bridge

A minimal SwiftUI app that exports HealthKit data to a user-selected folder.

## Features
- Requests HealthKit read permissions
- Lets you pick an export folder
- Exports HealthKit data for offline use

## Requirements
- Xcode 15+ (iOS 17+)
- A device or simulator with HealthKit available

## Build & Run
1. Open `iOS Health Bridge.xcodeproj` in Xcode.
2. Select a target device or simulator.
3. Build and run.

## Privacy
Health data is sensitive. Exports are stored only in the folder you select on your device.
